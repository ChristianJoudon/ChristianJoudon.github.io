Terran & the Wrath of the Wildwoods

A fast paced 2D rogue like wave defense game with hitbox precise combat and escalating bosses.

GodotGDScriptGame AI

Overview

Terran is a 2D rogue like built in Godot. Waves of enemies escalate toward boss fights, and combat is built around precise hitboxes and a set of abilities you manage under pressure.

What it does

  • A CharacterBody2D physics core with modular ability scenes balanced through cooldowns.
  • Signal driven enemy AI, randomized spawns, and JSON persisted progression between runs.

How it is built

Built in the Godot Engine with GDScript. A related design study, on dynamic environmental systems in RPGs, is written up in the essays.
