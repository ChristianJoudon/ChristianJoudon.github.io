UHnify: Campus Engagement

A student club platform for browsing clubs, managing memberships, and tracking events in real time.

Live site↗Project page↗
ReactMeteor.jsMongoDBNode.js

Overview

UHnify is a discovery platform for the University of Hawaiʻi at Mānoa club ecosystem, built to help students find and join clubs and keep up with events. It was a team project where I worked on the interface and the real time data layer.

What it does

  • A clean interface for browsing clubs, managing memberships, and viewing event calendars.
  • Real time synchronization so event details and memberships update live.

How it is built

React on the front end, Meteor.js and MongoDB for real time data and the backend, with Node.js underneath.
